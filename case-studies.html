Case Studies

Below you can explore some selected case studies developed during my Multimediadesign course.

Accessibility Project

This website below was developed during my first semester. The project was about creating an accessible website by applying the WCAG 2.1 guidelines. The project's primary focus was to design and develop a website that is inclusive and easy to use for all individuals, regardless of their physical or cognitive abilities. Key implementation areas included meeting criteria for color contrasts, using ARIA labels, implementing semantic HTML structure, and testing with accessibility tools.

It was a solo project. Topic was self-chosen.

Languages: HTML, CSS & JavaScript.

Link: http://theresepaulsen.dk

Branding Project

This website below was developed during my second semester. The project was about creating a digital branding campaign for our own imaginary multimedia agency. Key deliverables include a responsive, SEO-optimized WordPress website, a Brand Manual, as wel as multiple video and animation elements.

It was a group project. My responsibilities included development and deployment of the WordPress website, SEO-optimization, as well as creating the logo and animation shown below.

Softwares: Figma, WordPress, Adobe AfterEffects, Illustrator, InDesign.

Link: http://www.theresepaulsen.dk/wordpress/

Real-Life Project

This was a Real-Life project where we partnered with LadyBalance - a brand focused on intimate health - to understand their digital challenges and optimization wishes for their website. We delivered a WordPress solution focusing on improved UX, SEO, and WCAG accessibility. The goal of the project was to create a website, campaign and landing page with a significant "wow-effect" to strengthen their digital presence and differentiate them in the market.

It was a group project. My responsibilities included user-related research, user tests, SEO analysis, strategic analysis as well as developing the concept.

Softwares: Figma, WordPress.
